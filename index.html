    04 Personal - Jean Michel Basquiat

  • 디자인 의도 : 장 미쉘 바스키아에 대한 설명적 사이트이다. 그의 생애를 나타내고자 제작했다.
  • 기능 (Function) :
    1. 좌측 상단의 메뉴 아이콘을 누르면 메뉴 창이 나타나고 메뉴에 마우스를 올리면 색과 크기가 바뀌게 제작
    2. 각 메뉴를 누르면 해당 부분으로 바로 이동하도록 제작
    3. SAMO, CLUB 57/MUDD 부분의 이미지가 마우스를 이미지위에 올리고 스크롤하거나 클릭하면 전환되도록 제작
    4. Andy warhol 페이지에서 노란 글자를 누르면 숨겨진 글이 나오도록 제작
    5. FASHION/MUSIC 부분 노란 글자 옆의 빨간 아이콘을 누르면 숨겨진 글이 나오도록 제작
    6. DEAD 배너가 우->좌로 이동되도록 제작
    7. DEAD 글자를 누르면 숨겨진 글이 나오도록 제작
